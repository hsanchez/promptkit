# PromptKit

PromptKit is a small Git-based prompt versioning toolkit.

It assumes each repo owns its prompts:

```text
prompts/
  drafts/
    system.yaml.j2
  current/
    system.yaml
    metadata.json
  .vault/
    v0.1.0/
      system.yaml
      metadata.json
  promptspec.yaml
```

## Install locally

```bash
uv sync
```

## Commands

```bash
uv run prompt init
uv run prompt draft
uv run prompt check
uv run prompt release --patch
uv run prompt diff
uv run prompt rollback v0.1.0
```

## Semantics

```text
drafts/   = mutable prompt templates
current/  = active resolved prompt set
.vault/   = immutable prompt release history
```

Humans and agents edit only `prompts/drafts/`.

`prompts/current/` and `prompts/.vault/` are generated by PromptKit.
